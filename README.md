# Health-Care-Chatbot

## Overview
This is a Python-based healthcare chatbot that answers user queries using machine learning and predefined intents.

## Features
- Healthcare chatbot system
- GUI-based interaction
- Intent classification using NLP
- Model training support
- Data analysis using Jupyter Notebook

## Project Files
- chatbot_py.py → chatbot logic
- gui.py → graphical interface
- training_py.py → model training
- intents.json → chatbot responses
- heart-disease.csv → dataset
- Data_Analysis.ipynb → data analysis
